## chrisisler/cubetime

Simple cube timer.

- Click timer to start/stop
- Click scramble to re-scramble

### /api/solves

- [x] GET /api/solves - Read all Solves
- [x] POST /api/solves - Create a Solve
- [x] DELETE /api/solves - Delete all Solves
- [x] DELETE /api/solves/:id - Delete Solve by ID
